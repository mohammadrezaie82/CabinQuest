## About The Project

This is a Next.js project created with `create-next-app`.

The project includes a modern and scalable architecture using the latest features of Next.js. It is designed to provide a smooth development experience and optimized performance.

## Tech Stack

- Next.js
- React
- TypeScript
- Supabase
- Tailwind CSS
- React Query
- React Hook Form

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
